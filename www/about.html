About Me

Hello! I'm Chinwe Martins, a frontend developer who enjoys building clean and responsive websites. I come from a science background—Biochemistry to be specific—which helps me approach coding with structure and logic.

I started learning web development because I love how code can turn ideas into real, interactive things people can use. Over time, I’ve worked with HTML, CSS, JavaScript, and frameworks like React to build projects that not only work but also look good on all devices.

These days, I spend a lot of time exploring new frontend tools, designing layouts, and improving my skills. I enjoy working on real-world projects and learning from each one. When I’m not coding, I’m probably reading about tech trends or trying out a new design idea.
